# คำสั่งทั้งหมดของพิมพ์ดาว

---

##  หมวดหมู่ทั่วไป

| คำสั่ง  | รายละเอียด              |
| ------- | ----------------------- |
| `/help` | แสดงรายการคำสั่งทั้งหมด |
| `/ping` | เช็คสถานะของบอท         |

---

##  หมวดหมู่แอดมินทั่วไป

| คำสั่ง         | รายละเอียด                                      |
| -------------- | ----------------------------------------------- |
| `/admin-clear` | ล้างข้อความตามจำนวนที่ระบุ (สูงสุด 100 ข้อความ) |

---

##  หมวดหมู่มินิเกม

| คำสั่ง         | รายละเอียด                       |
| -------------- | -------------------------------- |
| `/g-hot`       | เช็คความฮ็อตของคุณ               |
| `/g-ship`      | คุณเข้ากันได้กับใครในเซิร์ฟเวอร์ |
| `/g-ทายเหรียญ` | ทายผลด้านหัวหรือก้อยของเหรียญ    |

---

## หมวดหมู่เครื่องเล่นเพลง

| คำสั่ง   | รายละเอียด           |
| -------- | -------------------- |
| `/play`  | เล่นเพลงจาก YouTube  |
| `/skip`  | ข้ามเพลง             |
| `/stop`  | หยุดเพลงและออกจากคิว |
| `/loop`  | วนเพลง               |
| `/queue` | เช็คคิวเพลง          |

---

##  หมวดหมู่บันทึกประวัติ (Logs)

| คำสั่ง              | รายละเอียด                                         |
| ------------------- | -------------------------------------------------- |
| `/logs-message-on`  | `[ + ]` เปิดใช้งานระบบบันทึกข้อความ (Message Logs) |
| `/logs-message-off` | `[ - ]` ปิดระบบบันทึกข้อความ                       |

---

##  หมวดหมู่ห้องเกม

| คำสั่ง             | รายละเอียด                          |
| ------------------ | ----------------------------------- |
| `/setup-g-flags`   | เซ็ตอัปห้องเกมทายชื่อประเทศจากธง    |
| `/setup-g-country` | เซ็ตอัปห้องเกมทายชื่อประเทศจากคำใบ้ |
| `/setup-g-city`    | เซ็ตอัปห้องเกมทายชื่อเมืองจากคำใบ้  |
| `/setup-g-count`   | เซ็ตอัปเกมนับเลข                    |

---

##  หมวดหมู่กดอิโมจิเพื่อรับบทบาท (Emoji Roles)

| คำสั่ง                     | รายละเอียด                                            |
| -------------------------- | ----------------------------------------------------- |
| `/emoji-role add`          | `[ 1 ]` ตั้งค่าระบบกดอิโมจิเพื่อรับบทบาท              |
| `/emoji-role-list`         | `[ * ]` ดู Emoji Roles ทั้งหมดในเซิร์ฟเวอร์           |
| `/emoji-role-single-emoji` | `[ * ]` กำหนดให้ข้อความนี้กดได้แค่ 1 อิโมจิ           |
| `/emoji-role-clear-all`    | `[ - ]` ล้างการตั้งค่าอิโมจิทุกอย่างในข้อความที่เลือก |
| `/emoji-role-remove`       | `[ - ]` ลบ Emoji Role ออกจากข้อความ                   |

---

##  หมวดหมู่อัตโนมัติเมื่อสมาชิกเข้า–ออกเซิร์ฟเวอร์

| คำสั่ง                | รายละเอียด                                                          |
| --------------------- | ------------------------------------------------------------------- |
| `/autorename-enable`  | `[ 1 ]` เปิด/ปิด ระบบเปลี่ยนชื่ออัตโนมัติเมื่อสมาชิกเข้าเซิร์ฟเวอร์ |
| `/autorename-setting` | `[ 2 ]` ตั้งค่าชื่ออัตโนมัติ<br>**ตัวแปร:** `{username} ชื่อผู้ใช้ เช่น พิมพ์ดาว` `{id} ไอดีผู้ใช้เช่น 12345678 `   **ตัวอย่างการตั้งค่า:** `{username}✨ (ตัวอย่าง)เมื่อผู้ใช้เข้าเซิฟจะได้รับการเปลี่ยนชื่อเป็น`  พิมพ์ดาว ✨  |
| `/role-auto-enable`   | `[ 1 ]` เปิด/ปิด ระบบมอบบทบาทอัตโนมัติ                              |
| `/role-auto-setting`  | `[ 2 ]` ตั้งค่าบทบาทที่จะมอบเมื่อสมาชิกเข้าร่วม                     |
| `/member-join-setup`  | ตั้งค่าระบบส่งข้อความต้อนรับสมาชิก                                  |
| `/member-left-setup`  | ตั้งค่าระบบส่งข้อความเมื่อสมาชิกออกจากเซิร์ฟเวอร์                   |
